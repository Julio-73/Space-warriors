# 🚀 SPACE WARRIORS - Edición Multiplataforma

## 🌟 Descripción
Juego épico de naves espaciales compatible con PC y Móvil creado con HTML5 Canvas.

## ✨ Características
- 📱 **100% Compatible con Móvil y PC**
- 🎮 **4 modos de juego** (Clásico, Supervivencia, Contrarreloj, Misión)
- 👾 **7 tipos de enemigos** con IA única
- 💫 **8 power-ups épicos** (misiles, rayo láser, congelar tiempo)
- 🏆 **Sistema de logros** desbloqueables
- 🎵 **Banda sonora espacial** generada proceduralmente
- 🌍 **5 fondos espaciales** diferentes
- 🕹️ **Controles táctiles** para móviles
- ⌨️ **Controles de teclado** para PC

## 🎯 Cómo Jugar

### 📱 Para Móvil:
- **Joystick virtual** - Mover nave
- **🔫 Botón rojo** - Disparar
- **⚡ Botón azul** - Turbo
- **🚀 Botón naranja** - Misiles
- **⏸️ Botón morado** - Pausa
- **🛡️ Botón dorado** - Escudo rápido

### 🖥️ Para PC:
- **WASD** - Mover nave
- **ESPACIO** - Disparar
- **SHIFT** - Turbo
- **M** - Disparar misiles
- **P** - Pausar juego

## 🛠️ Tecnologías
- HTML5 Canvas API
- CSS3 con animaciones avanzadas
- JavaScript ES6+
- Web Audio API
- Touch Events API
- Responsive Design
- GitHub Pages para hosting
